# moodtunes

A personal music mood journal that lets you log songs to your current mood, 
build mood-based playlists from your listening history, and discover new music 
that matches how you feel.

## Built with
- HTML, CSS, JavaScript
- Spotify Web API

## Features (in progress)
- Log songs by mood
- Auto-generated mood playlists
- Listening history timeline
- Music discovery based on your mood patterns
